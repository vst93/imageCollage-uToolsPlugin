# imageCollage-uToolsPlugin 图片拼接插件

如名，简单的图片垂直拼接，保存的图片在系统下载目录。

![UTOOLS_1619171271226.png](https://i.loli.net/2021/04/23/UaqvQuY4LeW5trd.png)

##### 快捷键
- `ctrl + a` `cmd + a` 添加图片
- `ctrl + d` `cmd + d` 清空图片
- `ctrl + s` `cmd + s` 保存图片
